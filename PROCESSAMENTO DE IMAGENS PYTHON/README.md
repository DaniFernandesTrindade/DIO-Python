## ✨ Funcionalidades

- 🖤 **Convert to Black & White** – converte uma imagem colorida para preto e branco
- 🔍 **Resize** – redimensiona imagens para qualquer tamanho
- 🌫️ **Blur** – aplica desfoque gaussiano com intensidade personalizada
